## Problem 1

给定平面上 {{< imath >}}2n{{< /imath >}} 个点，任意将 {{< imath >}}n{{< /imath >}} 个点染成红色，剩下 {{< imath >}}n{{< /imath >}} 个点染成蓝色。证明存在一种红蓝点的 {{< imath >}}1{{< /imath >}}-{{< imath >}}1{{< /imath >}} 配对，使得连接它们的 {{< imath >}}n{{< /imath >}} 条线段相互不相交。

**证**

定义红点集 {{< imath >}}R=\{ r_{1},r_{2},\dots,r_{n} \}{{< /imath >}}，蓝点集 {{< imath >}}B=\{ b_{1},b_{2},\dots,b_{n} \}{{< /imath >}}。{{< imath >}}R\to B{{< /imath >}} 的映射，对应一种配对方案，总数为 {{< imath >}}n!{{< /imath >}}。

对于任意一种配对方案 {{< imath >}}\sigma{{< /imath >}}，我们定义它的总长度 {{< imath >}}L(\sigma){{< /imath >}} 为该方案中所有线段的长度之和
{{< math >}}

L(\sigma) = \sum_{(r,b)\in\sigma}\text{dist}(r,b)

{{< /math >}}
由于总方案数有限，因此一定存在一种方案 {{< imath >}}\sigma_{m}{{< /imath >}} 使得总长度 {{< imath >}}L(\sigma_{m}){{< /imath >}} 最小。我们下面证明这个最小方案中没有相交的线段。

假设在 {{< imath >}}\sigma_{m}{{< /imath >}} 中存在两条线段相交，设为 {{< imath >}}(r_{1},b_{1}){{< /imath >}} 和 {{< imath >}}(r_{2},b_{2}){{< /imath >}}，相交于点 {{< imath >}}P{{< /imath >}}。那么我们考虑交换配对变为 {{< imath >}}(r_{1},b_{2}){{< /imath >}} 和 {{< imath >}}(r_{2},b_{1}){{< /imath >}}。根据三角不等式，有 {{< imath >}}\left| r_{1}b_{2} \right|<\left| r_{1}X \right|+\left| Xb_{2} \right|,\left| r_{2}b_{1} \right|<\left| r_{2}X \right|+\left| Xb_{1} \right|{{< /imath >}}，从而
{{< math >}}

\left| r_{1}b_{2} \right| +\left| r_{2}b_{1} \right| < \left| r_{1}X \right| +\left| Xb_{1} \right| +\left| r_{2}X \right| +\left| Xb_{2} \right| = \left| r_{1}b_{1} \right| +\left| r_{2}b_{2} \right| 

{{< /math >}}
发现交换后得到了一个总长度更小的方案，从而与总长度最小矛盾。这样我们就证明了总长度最小的配对方案所有线段必然不相交，证毕。

## Problem 2

给定一个 {{< imath >}}n{{< /imath >}} 个点的点集 {{< imath >}}V{{< /imath >}}，称一条线是好的当且仅当它恰好经过 {{< imath >}}V{{< /imath >}} 中的 {{< imath >}}3{{< /imath >}} 个点。

**(1)**

证明好的线段的数量至多为 {{< imath >}}n^{2} / 6{{< /imath >}}。

**证**

由于任意两个点确定一条直线，因此任意两条不同的直线不可能共用一个点对。一条好的线段恰好包含 {{< imath >}}3{{< /imath >}} 个点，从而恰好包含 {{< imath >}}3{{< /imath >}} 个点对。我们设好的线段数量为 {{< imath >}}m{{< /imath >}}，那么一共包含了 {{< imath >}}3m{{< /imath >}} 个不同的点对。又因为点对总数为 {{< imath >}}\binom{ n }{ 2 }{{< /imath >}}，因此
{{< math >}}

3m \leq  \binom{ n }{ 2 } \implies m \leq  \dfrac{n(n-1)}{6} < \dfrac{n^{2}}{6}

{{< /math >}}
**(2)**

设 {{< imath >}}M_{n}{{< /imath >}} 是 {{< imath >}}n{{< /imath >}} 个点所有可能的排布中最大的好的直线的数量。给出尽可能精确的 {{< imath >}}M_{n}{{< /imath >}} 的下界。

**解**

实际上 {{< imath >}}M_{n}{{< /imath >}} 的下界也是 {{< imath >}}\dfrac{n^{2}}{6}-O(n){{< /imath >}}。但是由于难以构造，这里给出一个也是 {{< imath >}}O(n^{2}){{< /imath >}}，为 {{< imath >}}\dfrac{n^{2}}{8}-O(n){{< /imath >}} 的构造。

我们利用三次函数来完成这个构造。构造函数 {{< imath >}}y=x^{3}{{< /imath >}}，取 {{< imath >}}x \in\{ -n, \dots,-1,0,1,\dots,n \}{{< /imath >}} 共 {{< imath >}}2n+1{{< /imath >}} 个点，对于每一个 {{< imath >}}x{{< /imath >}}，构造 {{< imath >}}P_{x}=(x,x^{3}){{< /imath >}}。下面我们证明这个点集满足条件。

根据三次函数的性质，我们知道任意一条直线与它至多只有三个交点，因此不可能出现四点共线的情况。并且利用代数性质容易证明三点共线的充要条件是横坐标之和为零，也就是若 {{< imath >}}x,y,z\in \{ -n, \dots, n \}{{< /imath >}} 满足 {{< imath >}}x+y+z=0{{< /imath >}}，那么 {{< imath >}}P_{x},P_{y},P_{z}{{< /imath >}} 共线。

在集合 {{< imath >}}\{ -n, \dots,n \}{{< /imath >}} 中，不互相同的 {{< imath >}}x+y+z=0{{< /imath >}} 的组合的数量等价于满足 {{< imath >}}-n\leq x+y\leq n{{< /imath >}} 的组数，从而求出总数为
{{< math >}}

3n^{2} + 3n - 6\lfloor n / 2 \rfloor 

{{< /math >}}
由于 {{< imath >}}x,y,z{{< /imath >}} 轮换，因此直线的总数为
{{< math >}}

\dfrac{1}{2}n^{2} + \dfrac{1}{2}n - \lfloor n / 2 \rfloor = \dfrac{(2n+1)^{2}}{8} - O(n)

{{< /math >}}

## Problem 3

设 {{< imath >}}\mathcal{F}{{< /imath >}} 为 {{< imath >}}[n]{{< /imath >}} 的一个相交子集族。证明存在一个包含 {{< imath >}}\mathcal{F}{{< /imath >}} 的相交子集族 {{< imath >}}\mathcal{F}'{{< /imath >}}，使得 {{< imath >}}|\mathcal{F}'| = 2^{n-1}{{< /imath >}}。

**证**

考虑 {{< imath >}}[n]{{< /imath >}} 的所有子集，共有 {{< imath >}}2^{n}{{< /imath >}} 个。我们将这 {{< imath >}}2^{n}{{< /imath >}} 个子集分成 {{< imath >}}2^{n-1}{{< /imath >}} 对互补的集合。也就是对于任意 {{< imath >}}A\subseteq[n]{{< /imath >}}，它和它的补集 {{< imath >}}A^{c}{{< /imath >}} 组成了一个互补对 {{< imath >}}\{ A,A^{c} \}{{< /imath >}}，显然 {{< imath >}}A{{< /imath >}} 和 {{< imath >}}A^{c}{{< /imath >}} 不相交。这说明在一个交族 {{< imath >}}\mathcal{F}{{< /imath >}}，它不可能同时包含 {{< imath >}}A{{< /imath >}} 和 {{< imath >}}A^{c}{{< /imath >}}。从而说明了一个交族的大小不可能超过互补对的数量，即 {{< imath >}}\left| \mathcal{F} \right|\leq 2^{n}-1{{< /imath >}}。

现在对于任意一个交族 {{< imath >}}\mathcal{F}\subseteq[n]{{< /imath >}}，我们进行以下操作来构造 {{< imath >}}\mathcal{F}'{{< /imath >}}：对于每一对互补对，如果 {{< imath >}}A\in \mathcal{F}{{< /imath >}} 或者 {{< imath >}}A^{c}\in \mathcal{F}{{< /imath >}}，那么我们保留它们在 {{< imath >}}\mathcal{F}'{{< /imath >}}，否则选择 {{< imath >}}A{{< /imath >}} 和 {{< imath >}}A^{c}{{< /imath >}} 中的一个加入 {{< imath >}}\mathcal{F}'{{< /imath >}}，需要保证加入后仍然保持了交族的性质。我们证明一定可以完成构造，假设某一步中我们无论加入 {{< imath >}}A{{< /imath >}} 还是 {{< imath >}}A^{c}{{< /imath >}} 都会破坏交族的性质，那么
- 不能加入 {{< imath >}}A{{< /imath >}} 说明 {{< imath >}}\mathcal{F}{{< /imath >}} 中存在集合 {{< imath >}}B{{< /imath >}} 使得 {{< imath >}}A\cap B=\emptyset{{< /imath >}}，从而 {{< imath >}}B\subseteq A^{c}{{< /imath >}}。
- 不能加入 {{< imath >}}A^{c}{{< /imath >}} 说明 {{< imath >}}\mathcal{F}{{< /imath >}} 中存在集合 {{< imath >}}C{{< /imath >}} 使得 {{< imath >}}A^{c}\cap C=\emptyset{{< /imath >}}，从而 {{< imath >}}C\subseteq A{{< /imath >}}。
如果两个情况同时发生，那么同时存在 {{< imath >}}B\subseteq A^{c}{{< /imath >}} 以及 {{< imath >}}C\subseteq A{{< /imath >}}，可以得到 {{< imath >}}B\cap C\subseteq A\cap A^{c}=\emptyset{{< /imath >}}，与 {{< imath >}}\mathcal{F}{{< /imath >}} 是一个交族矛盾！

因此对于每一个互补对，我们都可以选出一个集合加入 {{< imath >}}\mathcal{F}'{{< /imath >}}，所以 {{< imath >}}\left| \mathcal{F}' \right|=2^{n-1}{{< /imath >}}，与互补对的数量相等。

## Problem 4

假设我们可以将子图 {{< imath >}}K_{n}{{< /imath >}} 分解为 {{< imath >}}m{{< /imath >}} 个边不重叠的完全子图的并集，并且这些子图都和 {{< imath >}}K_{n}{{< /imath >}} 不同，证明：{{< imath >}}m\geq n{{< /imath >}}。

**证 1**

我们将图 {{< imath >}}K_n{{< /imath >}} 的顶点集 {{< imath >}}V{{< /imath >}} 视为几何中的点集，将分解出的 {{< imath >}}m{{< /imath >}} 个完全子图视为线集。我们将这个问题转化为满足 De Bruijn-Erdős 定理的结构。

对于任意两个顶点 {{< imath >}}u, v \in V{{< /imath >}}，它们在 {{< imath >}}K_n{{< /imath >}} 中由唯一的一条边连接。由于这些完全子图构成了边的划分，该边 {{< imath >}}(u, v){{< /imath >}} 必须恰好属于某一个唯一的完全子图。这对应了平面上两个点唯一确定一条直线。

题目已知这些完全子图均不同于 {{< imath >}}K_n{{< /imath >}}，这意味着没有任何一个子图包含了 {{< imath >}}V{{< /imath >}} 中的所有顶点，这对应了定理中要求的所有点不共线。

因此根据 De Bruijn-Erdős 定理，在线性空间中线的数量至少等于点的数量，我们就直接证明了 {{< imath >}}m\geq n{{< /imath >}}，证毕。
