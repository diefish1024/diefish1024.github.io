## Problem 1

假设有 {{< imath >}}m{{< /imath >}} 个俱乐部和 {{< imath >}}n{{< /imath >}} 个人，满足以下条件： (i) 每个俱乐部的人数为**偶数**； (ii) 任意两个俱乐部之间共同拥有的人数为**奇数**。 证明：{{< imath >}}m \le n{{< /imath >}}。

**证**：

我们在有限域 {{< imath >}}\mathbb{F}_{2}{{< /imath >}} 上考虑。设 {{< imath >}}v_{1},\dots,v_{m}\in \mathbb{F}_{2}^{n}{{< /imath >}} 为每个俱乐部对应向量。根据条件，我们知道
- 人数为偶数：{{< imath >}}v_{i}\cdot v_{i}=0{{< /imath >}}
- 交集为奇数：{{< imath >}}v_{i}\cdot v_{j}=1(i\neq j){{< /imath >}}

考察这 {{< imath >}}m{{< /imath >}} 个向量的线性相关性。假设存在系数 {{< imath >}}c_{1},\dots c_{m}\in \mathbb{F}_{2}{{< /imath >}} 使得
{{< math >}}

\sum_{i=1}^{m} c_{i}v_{i}=\mathbf{0}

{{< /math >}}
那么对于任意 {{< imath >}}j\in [m]{{< /imath >}}，我们将上式与 {{< imath >}}v_{j}{{< /imath >}} 做内积
{{< math >}}

\left( \sum_{i=1}^{m} c_{i}v_{i} \right)\cdot v_{j} = c_{j}(v_{j}\cdot v_{j}) + \sum_{i\neq j}c_{i}(v_{i}\cdot v_{j}) = 0

{{< /math >}}
带入得到
{{< math >}}

\sum_{i\neq j}c_{i}=0

{{< /math >}}
由于对于每个 {{< imath >}}j{{< /imath >}} 这个条件都要成立，从而 {{< imath >}}c_{j}=\sum_{i\in[m]}c_{i}{{< /imath >}}，这意味着所有的 {{< imath >}}c_{i}{{< /imath >}} 都相等，因此要么全为 {{< imath >}}0{{< /imath >}}，要么全为 {{< imath >}}1{{< /imath >}}。若全为 {{< imath >}}0{{< /imath >}}，则线性无关；若全为 {{< imath >}}1{{< /imath >}}，显然与 {{< imath >}}\sum_{i\neq j}c_{i}=0{{< /imath >}} 矛盾。

所以说明 {{< imath >}}v_{1},\dots,v_{m}{{< /imath >}} 线性无关，从而 {{< imath >}}\text{rank}(v_{1},\dots,v_{m})= m\leq n{{< /imath >}}（向量的维数为 {{< imath >}}n{{< /imath >}}，利用秩的基本性质）。

## Problem 2

假设有 {{< imath >}}m{{< /imath >}} 个俱乐部和 {{< imath >}}n{{< /imath >}} 个人，满足以下条件： (i) 每个俱乐部的人数为**奇数**； (ii) 任意两个俱乐部之间共同拥有的人数为**奇数**。 证明：{{< imath >}}m \le 2^{\lfloor(n-1)/2\rfloor}{{< /imath >}}。

**证**：

同样取每个俱乐部对应的向量 {{< imath >}}v_{1},\dots,v_{m}\in \mathbb{F}_{2}^{n}{{< /imath >}}。取第一个向量 {{< imath >}}v_{1}{{< /imath >}}，对于所有的 {{< imath >}}i\in[m]{{< /imath >}}，构造
{{< math >}}

u_{i} = (v_{i} + v_{1}) \in\mathbb{F}_{2}

{{< /math >}}
特别地，其中 {{< imath >}}u_{1}=v_{1}+v_{1}=\mathbf{0}{{< /imath >}}。我们考察 {{< imath >}}U=\{ u_{1},\dots,u_{m} \}{{< /imath >}}。

{{< imath >}}U{{< /imath >}} 中任意两个向量满足
{{< math >}}

\begin{align*}
u_{i}\cdot u_{j} & = (v_{i}+v_{1})(v_{j}+v_{1}) \\
 & = v_{i}\cdot v_{j} + v_{i}\cdot v_{1} + v_{j}\cdot v_{1} + v_{1}\cdot v_{1} \\
 & = 0\pmod 2
\end{align*}

{{< /math >}}
从而 {{< imath >}}U{{< /imath >}} 中任意两个向量的内积都是 {{< imath >}}0{{< /imath >}}，包括自己和自己的内积。从而 {{< imath >}}U{{< /imath >}} 张成的子空间 {{< imath >}}W=\text{span}\{ u_{2},\dots,u_{m} \}{{< /imath >}} 满足 {{< imath >}}W\subseteq W^{\perp}{{< /imath >}}。

除此之外，这些向量同样和 {{< imath >}}v_{1}{{< /imath >}} 正交：
{{< math >}}

u_{i}\cdot v_{1} = (v_{i}+v_{1})\cdot v_{1} = 0

{{< /math >}}
说明 {{< imath >}}W{{< /imath >}} 是 {{< imath >}}n{{< /imath >}} 维空间中与 {{< imath >}}v_{1}{{< /imath >}} 的正交的子空间，其中 {{< imath >}}v_{1}{{< /imath >}} 的正交补空间记为 {{< imath >}}V_{1}{{< /imath >}}，满足 {{< imath >}}\text{dim}(V_{1})=n-1{{< /imath >}}。

利用秩-零度定理，我们知道
{{< math >}}

\text{dim}(W) + \text{dim}(W^{\perp}) = \text{dim}(V_{1})=n-1

{{< /math >}}
由于 {{< imath >}}W\subseteq W^{\perp}{{< /imath >}}，因此
{{< math >}}

2\text{dim}(W) \leq  \text{dim}(W) + \text{dim}(W^{\perp}) = n-1

{{< /math >}}
从而
{{< math >}}

\text{dim}(W) \leq  \left\lfloor  \dfrac{n-1}{2}  \right\rfloor 

{{< /math >}}
由于这时 {{< imath >}}\mathbb{F}_{2}{{< /imath >}} 上的向量，每个自由维度只有个 {{< imath >}}2{{< /imath >}} 种可能。所以最多只可能有
{{< math >}}

m \leq  \left| W \right| \leq  2^{\lfloor (n-1)/2 \rfloor }

{{< /math >}}
个向量。

## Problem 3

设 {{< imath >}}A, B \subseteq \mathbb{R}^3{{< /imath >}} 是三维空间中的两个点集。已知 {{< imath >}}A{{< /imath >}} 中任意一点到 {{< imath >}}B{{< /imath >}} 中任意一点的距离都相等。 证明：{{< imath >}}\min\{|A|, |B|\} \le 2{{< /imath >}} （即 {{< imath >}}A{{< /imath >}} 和 {{< imath >}}B{{< /imath >}} 中至少有一个集合的元素个数不超过 {{< imath >}}2{{< /imath >}}）。

**证**：

设 {{< imath >}}A{{< /imath >}} 中的点为 {{< imath >}}\{ a_{1},a_{2},\dots, \}{{< /imath >}}，{{< imath >}}B{{< /imath >}} 中的点为 {{< imath >}}\{ b_{1},b_{2},\dots \}{{< /imath >}}。根据题意，存在一个常数 {{< imath >}}d{{< /imath >}} 使得对于任意 {{< imath >}}a\in A,b\in B{{< /imath >}}，满足 {{< imath >}}\| a-b \|^{2}=d^{2}{{< /imath >}}。

任取 {{< imath >}}a_{i},a_{j}\in A,b_{k},b_{l}\in B{{< /imath >}}（两个点不相等），那么
{{< math >}}

\begin{align*}
\| a_{i}-b_{k} \| ^{2} & = \| a_{i} \| ^{2} - 2a_{i}\cdot b_{k} + \| b_{k} \| ^{2} = d^{2} \\
\| a_{j}-b_{k} \| ^{2} & = \| a_{j} \|^{2} - 2a_{j}\cdot b_{k} + \| b_{k} \| ^{2}  =d^{2}
\end{align*}

{{< /math >}}
两式相减，得到
{{< math >}}

2(a_{i}-a_{j})\cdot b_{k} = \| a_{i} \| ^{2} - \| a_{j} \| ^{2}

{{< /math >}}
同理，对于 {{< imath >}}b_{l}{{< /imath >}}，也有
{{< math >}}

2(a_{i}-a_{j})\cdot b_{l} = \| a_{i} \| ^{2} - \| a_{j} \| ^{2}

{{< /math >}}
再次相减，就得到了
{{< math >}}

(a_{i}-a_{j})(b_{k}-b_{l})=0

{{< /math >}}

这说明 {{< imath >}}A{{< /imath >}} 中任意两点构成的向量垂直于 {{< imath >}}B{{< /imath >}} 中任意两点构成的向量。不妨设 {{< imath >}}A{{< /imath >}} 中任意两点构成的向量组成的空间为 {{< imath >}}V_{A}{{< /imath >}}，对应 {{< imath >}}B{{< /imath >}} 中任意两点之间的向量组成的空间为 {{< imath >}}V_{B}{{< /imath >}}，那么这说明 {{< imath >}}V_{A}\perp V_{B}{{< /imath >}}。

由于 {{< imath >}}V_{A}{{< /imath >}} 和 {{< imath >}}V_{B}{{< /imath >}} 都是 {{< imath >}}\mathbb{R}^{3}{{< /imath >}} 的子空间，并且它们正交，说明
{{< math >}}

\text{dim}(V_{A}) + \text{dim}(V_{B}) \leq  3

{{< /math >}}
如果 {{< imath >}}\left| A \right|\geq 3{{< /imath >}} 且 {{< imath >}}\left| B \right| \geq 3{{< /imath >}}，我们分类讨论。若 {{< imath >}}A{{< /imath >}} 中有三个点共线，此时根据简单几何知识，知道 {{< imath >}}B{{< /imath >}} 中不可能存在点到这三个点距离相等。

因此 {{< imath >}}A,B{{< /imath >}} 中都满足任意三点不共线。此时 {{< imath >}}V_{A}{{< /imath >}} 和 {{< imath >}}V_{B}{{< /imath >}} 至少都构成了一个平面，从而
{{< math >}}

\text{dim}(V_{A}) + \text{dim}(V_{B}) \geq  2 + 2 = 4

{{< /math >}}
矛盾！

因此假设不成立，{{< imath >}}A,B{{< /imath >}} 不可能同时拥有 {{< imath >}}3{{< /imath >}} 个及以上的点，即
{{< math >}}

\min\{ \left| A \right| ,\left| B \right|  \} \leq  2

{{< /math >}}
