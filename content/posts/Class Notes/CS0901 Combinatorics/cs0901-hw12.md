## Problem 1

对于随机图 {{< imath >}}\mathcal{G}(n, p){{< /imath >}}，属性“没有孤立点”具有一个锐阈值函数 {{< imath >}}p = \ln n / n{{< /imath >}}。

**证**

令 {{< imath >}}p=c\cdot \frac{\ln n}{n}{{< /imath >}}。设 {{< imath >}}X{{< /imath >}} 为图中孤立点的数量。
{{< math >}}
X = \sum_{i=1}^n I_i
{{< /math >}}
其中 {{< imath >}}I_i{{< /imath >}} 是指示变量：若顶点 {{< imath >}}i{{< /imath >}} 孤立，则 {{< imath >}}I_i=1{{< /imath >}}，否则 {{< imath >}}I_i=0{{< /imath >}}。我们需要计算 {{< imath >}}I_i{{< /imath >}} 的期望
{{< math >}}

\mathbf{E}[I_{i}] = (1-p)^{n-1}

{{< /math >}}
我们首先证明当 {{< imath >}}c>1{{< /imath >}} 时 {{< imath >}}\mathbb{P}(X > 0) \to 0{{< /imath >}}。根据 Markov 不等式，有
{{< math >}}

\mathbb{P}(X>0) = \mathbb{P}(X\geq  1) \leq  \mathbf{E}[X] = \sum_{i=1}^{n} \mathbf{E}[I_{i}] = n(1-p)^{n-1}

{{< /math >}}
使用不等式 {{< imath >}}1-x \le e^{-x}{{< /imath >}} 近似
{{< math >}}
\mathbf{E}[X] \le n e^{-p(n-1)} \approx n e^{-pn} = n e^{-c \ln n} = n \cdot n^{-c} = n^{1-c}
{{< /math >}}
因为 {{< imath >}}c > 1{{< /imath >}}，所以 {{< imath >}}1-c < 0{{< /imath >}}。当 {{< imath >}}n \to \infty{{< /imath >}} 时，{{< imath >}}n^{1-c} \to 0{{< /imath >}}。

所以 {{< imath >}}P(X > 0) \to 0{{< /imath >}}。

接着证明当 {{< imath >}}c< 1{{< /imath >}} 时 {{< imath >}}\mathbb{P}(X>0)\to 1{{< /imath >}}，也就是 {{< imath >}}\mathbb{P}(X=0)\to 0{{< /imath >}}。利用 Chebyshev 不等式，得到
{{< math >}}

\mathbb{P}(X=0) \leq  \frac{\text{Var}[X]}{\mathbf{E}[X]^{2}} = \frac{\mathbf{E}[X^{2}]}{\mathbf{E}[X]^{2}} - 1

{{< /math >}}
考虑证明 {{< imath >}}\frac{\mathbf{E}[X^{2}]}{\mathbf{E}[X]^{2}}\to 1{{< /imath >}}。计算 {{< imath >}}\mathbf{E}[X^{2}]{{< /imath >}}。首先
{{< math >}}

X^{2} = \left( \sum_{i=1}^{n} I_{i} \right)^{2}= \sum_{i=1}^{n} I_{i} + \sum_{i\neq j}I_{i}I_{j}

{{< /math >}}
那么
{{< math >}}

\mathbf{E}[X^{2}] = \mathbf{E}[X] + n(n-1)\mathbf{E}[I_{1}I_{2}]

{{< /math >}}
其中 {{< imath >}}\mathbf{E}[I_{1}I_{2}]{{< /imath >}} 等于两个顶点同时孤立的概率，这一共涉及了 {{< imath >}}2(n-2)+1=2n-3{{< /imath >}} 条边，因此
{{< math >}}

\mathbf{E}[I_{1}I_{2}] = (1-p)^{2n-3}

{{< /math >}}
带入就有
{{< math >}}

\frac{\mathbf{E}[X^{2}]}{\mathbf{E}[X]^{2}} = \frac{n(1-p)^{n-1}+n(n-1)(1-p)^{2n-3}}{n^{2}(1-p)^{2n-2}} \to \frac{1}{1-p}

{{< /math >}}
当 {{< imath >}}n\to \infty{{< /imath >}} 时有 {{< imath >}}p\to 0{{< /imath >}}，因此 {{< imath >}}\frac{1}{1-p}\to 1{{< /imath >}}，所以就有 {{< imath >}}\mathbb{P}(X=0)\to 0{{< /imath >}}，得证。

## Problem 2

我们将在接下来的 3 天里在全市举办冰淇淋派对。城里有很多酒吧。每个人都提交了一份愿望清单，形式如下
{{< math >}}
(P_1, d_1), (P_2, d_2), \dots, (P_{40}, d_{40})
{{< /math >}}
其中 {{< imath >}}P_i{{< /imath >}} 是不同的酒吧，且 {{< imath >}}d_i \in [3]{{< /imath >}}（即 {{< imath >}}d_i \in \{1, 2, 3\}{{< /imath >}}）。

如果一个人的愿望清单中至少有一个 {{< imath >}}(P_i, d_i){{< /imath >}} 满足“酒吧 {{< imath >}}P_i{{< /imath >}} 在第 {{< imath >}}d_i{{< /imath >}} 天举办派对”，那么这个人就是满意的。

这些愿望清单非常“友好”，每个酒吧最多出现在 2024 份清单上。

证明：我们可以安排派对（使得每个酒吧只在某一天举办一次派对），从而让所有提交清单的人都满意。

**证**

对每个酒吧，独立均匀地从 {{< imath >}}\{ 1,2,3 \}{{< /imath >}} 中随机选出一天举办配对，从而每个酒吧选定特定一天的概率是 {{< imath >}}\frac{1}{3}{{< /imath >}}。

设总共 {{< imath >}}n{{< /imath >}} 个人，对于第 {{< imath >}}i{{< /imath >}} 个人，定义坏事件 {{< imath >}}A_{i}{{< /imath >}} 为第 {{< imath >}}i{{< /imath >}} 个人不满意。某个坏事件发生当且仅当这个人的愿望全部落空。由于清单上的 {{< imath >}}40{{< /imath >}} 个酒吧是互不相同的，它们选日子的行为是相互独立的。因此，{{< imath >}}A_i{{< /imath >}} 发生的概率为：
{{< math >}}
P(A_i) = \left( \frac{2}{3} \right)^{40}
{{< /math >}}
我们将这个概率上界记为 {{< imath >}}p{{< /imath >}}。即 {{< imath >}}p = (2/3)^{40}{{< /imath >}}。

我们需要确定每个坏事件 {{< imath >}}A_i{{< /imath >}} 最多依赖于多少个其他的坏事件。这构成了依赖图的最大度数 {{< imath >}}\Delta{{< /imath >}}。由于每个酒吧至多出现在 {{< imath >}}2024{{< /imath >}} 个清单，并且每个人愿望有 {{< imath >}}40{{< /imath >}} 个酒吧，因此
{{< math >}}

\Delta \leq  40 \times 2024 = 80920

{{< /math >}}
利用对称 LLL 条件，带入数值可以验证验证
{{< math >}}

e\cdot p \cdot (\Delta+1) < 1

{{< /math >}}
从而一定存在方案能满足所有人。

## Problem 3

设 {{< imath >}}k \ge 10{{< /imath >}} 为一个整数，{{< imath >}}H = (V, E){{< /imath >}} 是一个 {{< imath >}}k{{< /imath >}}-均匀、{{< imath >}}k{{< /imath >}}-正则的超图。 请证明：{{< imath >}}H{{< /imath >}} 存在一个正常 {{< imath >}}2{{< /imath >}}-染色。

**证**

我们随机地对 {{< imath >}}H{{< /imath >}} 的每一个顶点进行染色。每个顶点染成红色或蓝色的概率均为 {{< imath >}}1/2{{< /imath >}}，且相互独立。对于超图中的每一条边 {{< imath >}}e \in E{{< /imath >}}，定义坏事件 {{< imath >}}A_e{{< /imath >}} 为边 {{< imath >}}e{{< /imath >}} 是单色的。显然
{{< math >}}

\mathbb{P}(A_{e}) = 2^{1-k}

{{< /math >}}
考虑一条边 {{< imath >}}e{{< /imath >}} 最多与多少条其他边有依赖关系，也就是和其他边相交。根据 {{< imath >}}k{{< /imath >}}-均匀和 {{< imath >}}k{{< /imath >}}-正则的性质，显然依赖图的最大度数 {{< imath >}}\Delta{{< /imath >}} 满足
{{< math >}}

\Delta \leq  k(k-1)

{{< /math >}}
从而利用对称版本的 LLL 条件，只需要验证
{{< math >}}

e\cdot 2^{1-k}\cdot(k(k-1)+1) \leq  1

{{< /math >}}
利用比值可以证明这关于 {{< imath >}}k{{< /imath >}} 在 {{< imath >}}k\geq 10{{< /imath >}} 时递减，并且带入 {{< imath >}}k=10{{< /imath >}}，满足
{{< math >}}

e\cdot 2^{-9}\cdot 91 < 1

{{< /math >}}
从而满足 LLL 条件，存在一种染色方案使得没有坏事件发生。

## Problem 4

范德瓦尔登数 {{< imath >}}W(2, k){{< /imath >}} 定义为使得“对集合 {{< imath >}}\{1, 2, \dots, n\}{{< /imath >}} 进行任意 {{< imath >}}2{{< /imath >}}-染色都必然包含一个长度为 {{< imath >}}k{{< /imath >}} 的单色等差数列”的最小整数 {{< imath >}}n{{< /imath >}}。 证明：{{< imath >}}W(2, k) > \frac{2^k}{2ek}{{< /imath >}}。

**证**

要证明 {{< imath >}}W(2, k) > N = \lfloor \frac{2^k}{2ek} \rfloor{{< /imath >}}，等价于证明对于 {{< imath >}}n = N{{< /imath >}}，存在一种 {{< imath >}}2{{< /imath >}}-染色方案，使得其中不包含任何长度为 {{< imath >}}k{{< /imath >}} 的单色等差数列。

对集合 {{< imath >}}\{1, 2, \dots, n\}{{< /imath >}} 中的每个数字随机染红或蓝，概率各为 {{< imath >}}1/2{{< /imath >}}。设 {{< imath >}}S{{< /imath >}} 是集合中的任意一个长度为 {{< imath >}}k{{< /imath >}} 的等差数列。定义坏事件 {{< imath >}}A_S{{< /imath >}} 为该等差数列是单色的。同样
{{< math >}}

p = \mathbb{P}(A_{S}) = 2^{1-k}

{{< /math >}}
考虑两个坏事件的依赖关系，这当且仅当两个等差数列至少共享一个数字。由于一个等差数列可以由首项和公差确定，而公差至多只有 {{< imath >}}n{{< /imath >}} 个取值，因此包含某个指定元素的等差数列至多只有 {{< imath >}}n{{< /imath >}} 个。这样就得到了与选定等差数列相交的等差数列数量的一个宽松上界 {{< imath >}}\Delta\leq k\cdot n{{< /imath >}}。

带入 LLL 条件，我们需要找到最大的 {{< imath >}}n{{< /imath >}} 使得
{{< math >}}

e\cdot p\cdot (\Delta+1) \leq  1

{{< /math >}}
带入就得到了
{{< math >}}

e\cdot 2^{1-k}\cdot kn \leq  1 \implies n\leq  \frac{1}{e\cdot k\cdot 2^{1-k}} = \frac{2^{k}}{2ek}

{{< /math >}}
从而只要 {{< imath >}}n\leq \frac{2^{k}}{2ek}{{< /imath >}} 成立，就必然存在某种染色方案使得没有单色等差数列，因此这就证明了
{{< math >}}

W(2,k) > \frac{2^{k}}{2ek}

{{< /math >}}
