## Problem 1

设 {{< imath >}}n>rst{{< /imath >}}。证明 {{< imath >}}n{{< /imath >}} 个数的实数列一定至少含有长度为 {{< imath >}}r{{< /imath >}} 的严格上升子序列、长度为 {{< imath >}}s{{< /imath >}} 的严格下降子序列、长度为 {{< imath >}}t{{< /imath >}} 的常数列中的一个。

**证**

反证法，假设结论不成立。

我们为第 {{< imath >}}i{{< /imath >}} 个元素 {{< imath >}}x_{i}{{< /imath >}} 记下标签 {{< imath >}}(a_{i},b_{i}){{< /imath >}}，其中 {{< imath >}}a_{i}{{< /imath >}} 表示以 {{< imath >}}x_{i}{{< /imath >}} 结尾的最长上升子序列长度，{{< imath >}}b_{i}{{< /imath >}} 表示以 {{< imath >}}x_{i}{{< /imath >}} 结尾的最长下降子序列长度。那么我们有 {{< imath >}}a_{i}\in[r-1],b_{i}\in[s-1]{{< /imath >}}。因此总共只有 {{< imath >}}(r-1)(s-1){{< /imath >}} 种标签，此时又有 {{< imath >}}n>t\cdot(r-1)(s-1){{< /imath >}}，所以至少有 {{< imath >}}t{{< /imath >}} 个元素的标签完全相同。

我们假设 {{< imath >}}a_{i}=a_{j},b_{i}=b_{j}{{< /imath >}}，可以证明 {{< imath >}}x_{i}=x_{j}{{< /imath >}}。如果不相等，不妨设 {{< imath >}}i< j{{< /imath >}}。若 {{< imath >}}x_{i}< x_{j}{{< /imath >}}，那么我们将 {{< imath >}}x_{j}{{< /imath >}} 加到以 {{< imath >}}x_{i}{{< /imath >}} 结尾的最长上升子序列之后，就得到了一个长度为 {{< imath >}}a_{i}+1{{< /imath >}} 的上升子序列，所以 {{< imath >}}a_{j}\geq a_{i}+1{{< /imath >}}，与 {{< imath >}}a_{i}=a_{j}{{< /imath >}} 矛盾；若 {{< imath >}}x_{i}>x_{j}{{< /imath >}}，同理可以得到 {{< imath >}}b_{j}\geq b_{i}+1{{< /imath >}}，与 {{< imath >}}b_{i}=b_{j}{{< /imath >}} 矛盾。所以一定有 {{< imath >}}x_{i}=x_{j}{{< /imath >}}。

所以这 {{< imath >}}t{{< /imath >}} 个标签完全相同的元素本身也相同，我们就得到了一个长度为 {{< imath >}}t{{< /imath >}} 的常数列，从而和我们的假设矛盾。证毕！

## Problem 2

证明对于任意正整数 {{< imath >}}c{{< /imath >}}，存在 {{< imath >}}N=N(c){{< /imath >}} 满足对于 {{< imath >}}[N]{{< /imath >}} 所有子集的任意的 {{< imath >}}c{{< /imath >}}-染色，存在不相交的非空集合 {{< imath >}}X,Y{{< /imath >}} 满足 {{< imath >}}X,Y{{< /imath >}} 和 {{< imath >}}X\cup Y{{< /imath >}} 具有相同的颜色。

**证**

我们取 {{< imath >}}N(c)=R_{c}(3;2){{< /imath >}}。对 {{< imath >}}[N]{{< /imath >}} 是所有子集任意 {{< imath >}}c{{< /imath >}}-染色后，对于 {{< imath >}}K_{N}{{< /imath >}} 中的任意一条边 {{< imath >}}(i,j){{< /imath >}}，不妨设 {{< imath >}}i< j{{< /imath >}}，我们将它的颜色定义为集合 {{< imath >}}\{ i, i+1,\dots,j-1\}{{< /imath >}} 的颜色。根据 Ramsey 定理，我们知道其中必然存在一个同色三角形，不妨设三个顶点为 {{< imath >}}u,v,w{{< /imath >}}，满足 {{< imath >}}u< v< w{{< /imath >}}，我们就知道了
{{< math >}}

\{ u, \dots,v-1 \},\{ v, \dots,w-1 \}, \{ u, \dots, w-1 \}

{{< /math >}}
三个集合颜色相同，于是直接取
{{< math >}}

\begin{align*}
X & = \{ u, \dots,v-1 \} \\
Y & = \{ v, \dots,w-1 \} \\
X \cup Y & = \{ u, \dots, w-1 \}
\end{align*}

{{< /math >}}
满足 {{< imath >}}X,Y,X\cup Y{{< /imath >}} 具有相同的颜色。

## Problem 3

对于任何正整数 {{< imath >}}c{{< /imath >}} 和 {{< imath >}}l{{< /imath >}}，存在一个数 {{< imath >}}w = w(c, l){{< /imath >}}，使得对集合 {{< imath >}}[w] = \{1, 2, ..., w\}{{< /imath >}} 的任意 {{< imath >}}c{{< /imath >}}-着色，都存在一个单色的集合，其形式为 {{< imath >}}\{a, a + d, a + 2d, ..., a + ld\} \cup \{d\}{{< /imath >}}，其中 {{< imath >}}a{{< /imath >}} 和 {{< imath >}}d{{< /imath >}} 为某个正整数。

**证**

我们将对颜色数 {{< imath >}}c{{< /imath >}} 使用数学归纳法来证明。

当 {{< imath >}}c=1{{< /imath >}}，只有一种颜色，集合 {{< imath >}}[w]{{< /imath >}} 中的所有整数都是同色的。我们需要找到一个单色的集合 {{< imath >}}\{a, a+d, ..., a+ld\} \cup \{d\}{{< /imath >}}。我们可以简单地取 {{< imath >}}d=1{{< /imath >}} 和 {{< imath >}}a=1{{< /imath >}}。这样我们得到的集合是 {{< imath >}}\{1, 1, 1+1, 1+2, ..., 1+l\} = \{1, 2, ..., l+1\}{{< /imath >}}。为了确保这个集合包含在 {{< imath >}}[w]{{< /imath >}} 中，我们只需令 {{< imath >}}w(1, l) = l+1{{< /imath >}}。此时集合 {{< imath >}}\{1, 2, ..., l+1\}{{< /imath >}} 中的所有元素都在 {{< imath >}}[l+1]{{< /imath >}} 中，并且都是单色的。因此定理在 {{< imath >}}c=1{{< /imath >}} 时成立。

假设定理对于 {{< imath >}}c-1{{< /imath >}} 种颜色成立，即我们假设 {{< imath >}}w(c-1, l){{< /imath >}} 存在。现在我们需要证明定理对于 {{< imath >}}c{{< /imath >}} 种颜色也成立。

令 {{< imath >}}w' = w(c-1, l){{< /imath >}}。根据归纳假设，这个数是存在的。令 {{< imath >}}W = W(c, l \cdot w' + 1){{< /imath >}}，其中 {{< imath >}}W(c, k){{< /imath >}} 保证了任何对 {{< imath >}}[W]{{< /imath >}} 的 {{< imath >}}c{{< /imath >}}-着色都包含一个长度为 {{< imath >}}k{{< /imath >}} 的单色等差数列。我们将证明 {{< imath >}}w(c, l) \le W{{< /imath >}}。考虑对集合 {{< imath >}}[W]{{< /imath >}} 的任意一个 {{< imath >}}c{{< /imath >}}-着色 {{< imath >}}\chi{{< /imath >}}。根据定义，在 {{< imath >}}[W]{{< /imath >}} 中必定存在一个长度为 {{< imath >}}l \cdot w' + 1{{< /imath >}} 的单色等差数列，我们设为 {{< imath >}}A = \{b, b+d', b+2d', ..., b+(l \cdot w')d'\}{{< /imath >}}，并假设它的颜色为 {{< imath >}}C_1{{< /imath >}}。

现在我们考虑公差 {{< imath >}}d'{{< /imath >}} 的颜色 {{< imath >}}\chi(d'){{< /imath >}}。分以下几种情况讨论

若 {{< imath >}}\chi(d') = C_1{{< /imath >}}，在这种情况下，我们令 {{< imath >}}a = b{{< /imath >}} 且 {{< imath >}}d = d'{{< /imath >}}。那么集合 {{< imath >}}\{d, a, a+d, ..., a+ld\}{{< /imath >}} 就等于 {{< imath >}}\{d', b, b+d', ..., b+ld'\}{{< /imath >}}。因为 {{< imath >}}\{b, b+d', ..., b+ld'\}{{< /imath >}} 是等差数列 {{< imath >}}A{{< /imath >}} 的一个子集，它们的颜色都是 {{< imath >}}C_1{{< /imath >}}。同时我们已知 {{< imath >}}\chi(d') = C_1{{< /imath >}}。因此，我们找到了一个颜色为 {{< imath >}}C_1{{< /imath >}} 的目标单色集合。

若 {{< imath >}}\chi(d') \ne C_1{{< /imath >}} ，虑集合 {{< imath >}}S = \{d', 2d', 3d', ..., w'd'\}{{< /imath >}}。

- 如果在集合 {{< imath >}}S{{< /imath >}} 中，存在某个元素 {{< imath >}}kd'{{< /imath >}} 的颜色为 {{< imath >}}C_1{{< /imath >}}，即 {{< imath >}}\chi(kd')=C_1{{< /imath >}}。那么我们令 {{< imath >}}d = kd'{{< /imath >}} 且 {{< imath >}}a = b{{< /imath >}}。考虑集合 {{< imath >}}\{d, a, a+d, ..., a+ld\} = \{kd', b, b+kd', ..., b+lkd'\}{{< /imath >}}。由于 {{< imath >}}k \le w'{{< /imath >}}，数列 {{< imath >}}\{b, b+kd', ..., b+lkd'\}{{< /imath >}} 是原始等差数列 {{< imath >}}A{{< /imath >}} 的一个子数列（因为 {{< imath >}}lk \le lw'{{< /imath >}}），因此它的所有元素颜色都是 {{< imath >}}C_1{{< /imath >}}。又因为 {{< imath >}}\chi(d)=\chi(kd')=C_1{{< /imath >}}，我们再次找到了一个颜色为 {{< imath >}}C_1{{< /imath >}} 的目标单色集合。

- 如果对于所有的 {{< imath >}}k \in \{1, 2, ..., w'\}{{< /imath >}}，颜色 {{< imath >}}\chi(kd'){{< /imath >}} 都不是 {{< imath >}}C_1{{< /imath >}}。这意味着对集合 {{< imath >}}S=\{d', 2d', ..., w'd'\}{{< /imath >}} 的着色 {{< imath >}}\chi{{< /imath >}} 只使用了 {{< imath >}}c-1{{< /imath >}} 种颜色（颜色 {{< imath >}}C_1{{< /imath >}} 被排除了）。现在我们在集合 {{< imath >}}[w']=\{1, 2, ..., w'\}{{< /imath >}} 上定义一个新的着色 {{< imath >}}\chi'{{< /imath >}}，规则为 {{< imath >}}\chi'(k) = \chi(k \cdot d'){{< /imath >}}。这个新的着色 {{< imath >}}\chi'{{< /imath >}} 最多只使用 {{< imath >}}c-1{{< /imath >}} 种颜色。根据我们的归纳假设，在 {{< imath >}}[w']{{< /imath >}} 上进行 {{< imath >}}\chi'{{< /imath >}} 着色，必定存在一个单色的集合 {{< imath >}}\{d^*, a^*, a^*+d^*, ..., a^*+ld^*\}{{< /imath >}}。假设这个集合在 {{< imath >}}\chi'{{< /imath >}} 着色下的颜色为 {{< imath >}}C_2{{< /imath >}} (其中 {{< imath >}}C_2 \ne C_1{{< /imath >}})。根据 {{< imath >}}\chi'{{< /imath >}} 的定义，这意味着
{{< math >}}

\begin{align*}
 \chi(d^{*}\cdot d') & = C_{2} \\
 \chi(a^{*}\cdot d') & = C_{2} \\
 \chi((a^{*}+d^{*})\cdot d') & = \chi(a^{*}d' + d^{*}d') = C_{2} \\
 & \dots \\
\chi((a^*+ld^*) \cdot d') & = \chi(a^*d' + l(d^*d')) = C_2
\end{align*}

{{< /math >}}
现在，我们令 {{< imath >}}d = d^*d'{{< /imath >}} 且 {{< imath >}}a = a^*d'{{< /imath >}}。那么我们就找到了一个在原始着色 {{< imath >}}\chi{{< /imath >}} 下的单色集合 {{< imath >}}\{d, a, a+d, ..., a+ld\}{{< /imath >}}，其颜色为 {{< imath >}}C_2{{< /imath >}}。

在所有可能的情况下，我们都能够找到所要求的单色结构。因此，归纳步骤完成，定理得证。

## Problem 4

证明任何图 {{< imath >}}G{{< /imath >}} 至少有 {{< imath >}}\binom{\chi(G)}{2}{{< /imath >}} 条边。

**证**

设 {{< imath >}}k=\chi(G){{< /imath >}}，我们使用 {{< imath >}}k{{< /imath >}} 中颜色 {{< imath >}}\{ c_{1},c_{2},\dots,c_{k} \}{{< /imath >}} 对 {{< imath >}}G{{< /imath >}} 进行染色。每种颜色的点都形成了一个非空独立集，对于颜色 {{< imath >}}i{{< /imath >}}，我们称所有颜色为 {{< imath >}}c_{i}{{< /imath >}} 的点的集合为颜色类 {{< imath >}}V_{i}{{< /imath >}}。对于两个不同的颜色类 {{< imath >}}V_{i},V_{j}{{< /imath >}}，我们证明 {{< imath >}}V_{i},V_{j}{{< /imath >}} 之间至少有一条边，否则合并 {{< imath >}}V_{i},V_{j}{{< /imath >}} 可以得到一个更大的独立集，从而可以只用 {{< imath >}}k-1{{< /imath >}} 中颜色染色，与最小染色矛盾。从而一共 {{< imath >}}k=\chi(G){{< /imath >}} 个颜色类，至少需要 {{< imath >}}\binom{ \chi(G) }{ 2 }{{< /imath >}} 条边，因为两两之间都需要有边相连。 

## Problem 5

假设 {{< imath >}}G = (V, E){{< /imath >}} 是一个包含 {{< imath >}}n{{< /imath >}} 个顶点和 {{< imath >}}m{{< /imath >}} 条边的图。对于一条边 {{< imath >}}e = \{u, v\}{{< /imath >}}，令 {{< imath >}}t(e){{< /imath >}} 为包含 {{< imath >}}e{{< /imath >}} 的三角形数量，并令 {{< imath >}}t(G){{< /imath >}} 为图 {{< imath >}}G{{< /imath >}} 中的三角形总数。

1. 证明对于每一对相邻顶点 {{< imath >}}(u, v){{< /imath >}} ，{{< imath >}}\deg(u) + \deg(v) - t(\{u, v\}) \leq n{{< /imath >}} 。
2. 证明 {{< imath >}}t(G) \geq \frac{m}{3n}(4m - n^2){{< /imath >}} 。

**证**

**(1)**

设 {{< imath >}}N(u){{< /imath >}} 表示与 {{< imath >}}u{{< /imath >}} 相邻的点的集合，{{< imath >}}t(\{ u,v \}){{< /imath >}} 即为同时与 {{< imath >}}u,v{{< /imath >}} 相邻的点的个数，为 {{< imath >}}\left| N(u)\cap N(v) \right|{{< /imath >}}，因此我们有
{{< math >}}

\left| N(u)\cup N(v) \right| = \left| N(u) \right| + \left| N(v) \right| - \left| N(u)\cap N(v) \right| = \text{deg}(u) + \text{deg}(v) - t(\{ u,v \})

{{< /math >}}
显然 {{< imath >}}\left| N(u)\cup N(v) \right|\leq \left| V \right|=n{{< /imath >}}，于是就得到了
{{< math >}}

\deg(u) + \deg(v) - t(\{u, v\}) \leq n

{{< /math >}}
**(2)**

根据 {{< imath >}}(1){{< /imath >}}，我们得到
{{< math >}}

t(\{ u,v \}) \geq  \text{deg}(u) + \text{deg}(v) - n

{{< /math >}}
我们考虑 {{< imath >}}\sum_{e\in E}t(e){{< /imath >}}，每个三角形包含 {{< imath >}}3{{< /imath >}} 条边，会被统计 {{< imath >}}3{{< /imath >}} 次，因此
{{< math >}}

\sum_{e\in E}t(e) = 3t(G)

{{< /math >}}
同时左侧我们有
{{< math >}}

\begin{align*}
\sum_{e\in E}t(e) & = \sum_{\{ u,v \}\in E}t(\{ u,v \}) \\
 & \geq  \sum_{\{ u,v \}\in E}(\text{deg}(u)+\text{deg}(v) - n) \\
 & = \sum_{u\in V}\text{deg}^{2}(u) - mn
\end{align*}

{{< /math >}}
其中利用 Cauchy 不等式，
{{< math >}}

\begin{align*}
\left( \sum_{u\in V}1^{2} \right)\left( \sum_{u\in V}\text{deg}^{2}(u) \right)  & \leq  \left( \sum_{u\in V}\text{deg}(u) \right)^{2} \\
& = 4m^{2} \\
\implies \sum_{u\in V}\text{deg}^{2}(u) & \leq  \dfrac{4m^{2}}{\sum_{u\in V}1} = \dfrac{4m^{2}}{n}
\end{align*}

{{< /math >}}
带入得到
{{< math >}}

t(G) \leq  \dfrac{1}{3}\left( \dfrac{4m^{2}}{n} - mn \right) = \dfrac{m}{3n}(4m - n^{2})

{{< /math >}}
