## Problem 1

图 {{< imath >}}G=(V,E){{< /imath >}} 有 {{< imath >}}n{{< /imath >}} 个顶点和 {{< imath >}}m{{< /imath >}} 条边，图 {{< imath >}}H{{< /imath >}} 是较小的图，且 {{< imath >}}G{{< /imath >}} 不包含 {{< imath >}}H{{< /imath >}} 作为子图。证明当 {{< imath >}}k > n^2 \ln n / m{{< /imath >}} 时，完全图 {{< imath >}}K_n{{< /imath >}} 存在一种边 {{< imath >}}k{{< /imath >}}-染色方案，使得 {{< imath >}}K_n{{< /imath >}} 中不存在单色的 {{< imath >}}H{{< /imath >}}。

**证**

令 {{< imath >}}V{{< /imath >}} 为 {{< imath >}}K_{n}{{< /imath >}} 的顶点集。随机选取 {{< imath >}}k{{< /imath >}} 个 {{< imath >}}V{{< /imath >}} 上的随机排列，将 {{< imath >}}G{{< /imath >}} 映射到 {{< imath >}}V{{< /imath >}}，得到与 {{< imath >}}G{{< /imath >}} 同构的 {{< imath >}}k{{< /imath >}} 个图 {{< imath >}}G_{1},\dots,G_{k}{{< /imath >}}。

考虑 {{< imath >}}K_{n}{{< /imath >}} 中的任意一条边 {{< imath >}}e{{< /imath >}} 未被覆盖的概率。对于任意一个图，{{< imath >}}e{{< /imath >}} 在这个图中的概率为
{{< math >}}

p = \mathbb{P}[e\in E(G_{i})] = \frac{m}{\binom{ n }{ 2 } }

{{< /math >}}
从而
{{< math >}}

\mathbb{P}\left[ e\not\in \bigcup_{i=1}^{k}E(G_{i}) \right] = (1-p)^{k} = \left( 1-\frac{m}{\binom{ n }{ 2 } } \right)^{k}

{{< /math >}}
我们证明存在一种方案使得 {{< imath >}}K_{n}{{< /imath >}} 的所有边都被覆盖，等价于证明至少有一条边未被覆盖的概率小于 {{< imath >}}1{{< /imath >}}。设事件 {{< imath >}}A{{< /imath >}} 表示至少有一条边未被覆盖，那么根据 Union Bound，有
{{< math >}}

\mathbb{P}[A] \leq  \sum_{e\in E(K_{n})}\mathbb{P}\left[ e\not\in \bigcup_{i=1}^{k}E(G_{i}) \right] = \binom{ n }{ 2 } \left( 1-\frac{m}{\binom{ n }{ 2 } } \right)^{k}

{{< /math >}}
带入不等式 {{< imath >}}1-x< e^{ -x }{{< /imath >}}，我们尝试证明
{{< math >}}

\binom{ n }{ 2 } \exp\left( -\frac{mk}{\binom{ n }{ 2 } } \right) < 1

{{< /math >}}
化简就得到了在
{{< math >}}

k > \frac{\binom{ n }{ 2 }}{m}\ln \binom{ n }{ 2 } 

{{< /math >}}
时不等式成立。

由于
{{< math >}}

k > \frac{n^{2}}{m}\ln n = \frac{n^{2} / 2}{m}\ln n^{2} > \frac{\binom{ n }{ 2 } }{m}\ln \binom{ n }{ 2 } 

{{< /math >}}
因此必有 {{< imath >}}\mathbb{P}[A] < 1{{< /imath >}}，从而一定存在一组副本 {{< imath >}}G_1, \dots, G_k{{< /imath >}}，它们的并集覆盖了 {{< imath >}}K_n{{< /imath >}} 的所有边。于是我们定义我们定义边染色如下：对于 {{< imath >}}K_n{{< /imath >}} 中的每条边 {{< imath >}}e{{< /imath >}}，定义其颜色 {{< imath >}}c(e){{< /imath >}} 为覆盖它的最小的副本索引
{{< math >}}
c(e) = \min \{i \mid e \in E(G_i)\}
{{< /math >}}
在这种染色方案下，对于任意颜色 {{< imath >}}i{{< /imath >}}，所有颜色为 {{< imath >}}i{{< /imath >}} 的边组成的集合 {{< imath >}}E_i{{< /imath >}} 都是 {{< imath >}}E(G_i){{< /imath >}} 的子集，即 {{< imath >}}E_i \subseteq E(G_i){{< /imath >}}。因为 {{< imath >}}G_i \cong G{{< /imath >}}，而已知 {{< imath >}}G{{< /imath >}} 不包含 {{< imath >}}H{{< /imath >}} 作为子图，所以 {{< imath >}}G_i{{< /imath >}} 也不包含 {{< imath >}}H{{< /imath >}}，因此，它的子集 {{< imath >}}E_i{{< /imath >}} 也不可能包含 {{< imath >}}H{{< /imath >}}。

综上所述，{{< imath >}}K_n{{< /imath >}} 的这种 {{< imath >}}k{{< /imath >}}-边染色中不包含单色的 {{< imath >}}H{{< /imath >}}。得证。
## Problem 2

证明对于任意 {{< imath >}}r{{< /imath >}} 和 {{< imath >}}n{{< /imath >}}（为简化假设 {{< imath >}}n{{< /imath >}} 能被 {{< imath >}}r{{< /imath >}} 整除），存在一个定义在 {{< imath >}}[n]{{< /imath >}} 上的 {{< imath >}}r{{< /imath >}}-一致超图 {{< imath >}}\mathcal{H}{{< /imath >}}，至少有 {{< imath >}}\binom{n}{r}/e^r{{< /imath >}} 条超边，其独立数满足
{{< math >}}
\alpha(\mathcal{H}) \geq n(1 - \frac{1}{r})
{{< /math >}}
超图的独立集是指不包含任何完整超边的顶点子集。

**证**

我们将 {{< imath >}}n{{< /imath >}} 个顶点均分成 {{< imath >}}r{{< /imath >}} 个组，每组大小均为 {{< imath >}}\frac{n}{r}{{< /imath >}}。定义 {{< imath >}}\mathcal{H}{{< /imath >}} 的边集 {{< imath >}}E{{< /imath >}} 满足一个 {{< imath >}}r{{< /imath >}} 元组 {{< imath >}}\{ v_{1},v_{2},\dots,v_{n} \}{{< /imath >}} 为一条边当且仅当每个 {{< imath >}}v_{i}{{< /imath >}} 来自不同的组。这样我们任意删掉一组边，剩下的顶点数为
{{< math >}}

n - \left| V_{1} \right| = n\left( 1-\frac{1}{r} \right)

{{< /math >}}
构成了一个独立集。从而这个构造的独立数满足要求。

下面我们验证这个构造的边数符合要求。由于每组可以任取一个点形成边，因此
{{< math >}}

\left| E \right| = \left| V_{1} \right| ^{r} = \left( \frac{n}{r} \right)^{r}

{{< /math >}}
因此我们只需要证明
{{< math >}}

\left( \frac{n}{r} \right)^{r} > \dfrac{\binom{ n }{ r } }{e^{ r }}

{{< /math >}}
 变形后等价于证明
 {{< math >}}

\binom{ n }{ r }  < \left( \frac{ne}{r} \right)^{r}

{{< /math >}}
由于
{{< math >}}

e^{ r } = 1 + r + \dots + \frac{r^{r}}{r!} + \dots > \frac{r^{r}}{r!} \implies \frac{1}{r!} < \frac{e^{r}}{r^{r}}

{{< /math >}}
那么
{{< math >}}

\binom{ n }{ r } = \frac{n^{\underline{r}}}{r!} < \frac{n^{r}}{r!} < \frac{e^{r}}{r^{r}}\cdot n^{r} = \left( \frac{ne}{r} \right)^{r}

{{< /math >}}
证毕。

## Problem 3

证明对于任意常数 {{< imath >}}c_1 > 0{{< /imath >}}，都存在另一个常数 {{< imath >}}c_2 > 0{{< /imath >}}，使得对于任意包含 {{< imath >}}n{{< /imath >}} 个顶点、{{< imath >}}m{{< /imath >}} 条超边的 {{< imath >}}3{{< /imath >}}-一致超图 {{< imath >}}\mathcal{H}{{< /imath >}}，只要满足 {{< imath >}}m \geq c_1 n{{< /imath >}}，就有
{{< math >}}
\alpha(\mathcal{H}) \geq \frac{c_2 n \sqrt{n}}{\sqrt{m}}
{{< /math >}}
**证**

设 {{< imath >}}\mathcal{H} = (V, E){{< /imath >}} 是一个 {{< imath >}}3{{< /imath >}}-一致超图，顶点数 {{< imath >}}|V|=n{{< /imath >}}，边数 {{< imath >}}|E|=m{{< /imath >}}。 构造一个随机顶点子集 {{< imath >}}S \subseteq V{{< /imath >}}，其中每一个顶点 {{< imath >}}v \in V{{< /imath >}} 被选入 {{< imath >}}S{{< /imath >}} 的概率都是 {{< imath >}}p{{< /imath >}}，且各顶点是否被选是相互独立的。

令随机变量 {{< imath >}}X = |S|{{< /imath >}} 表示 {{< imath >}}S{{< /imath >}} 中的顶点数量，随机变量 {{< imath >}}Y{{< /imath >}} 表示 {{< imath >}}S{{< /imath >}} 中包含的完整超边的数量。计算它们的期望
{{< math >}}

\mathbf{E}[X] = \sum_{v\in V}\mathbb{P}[v\in S] = n\cdot p

{{< /math >}}
由于一条超边包含三个独立选择的顶点，因此 {{< imath >}}\mathbb{P}[e\subseteq S]=p^{3}{{< /imath >}}，从而
{{< math >}}

\mathbf{E}[Y] = \sum_{e\in E}\mathbb{P}[e\subseteq S] =m\cdot p^{3}

{{< /math >}}
我们现在考虑将随机选择的 {{< imath >}}S{{< /imath >}} “修剪”成一个独立集。对于 {{< imath >}}S{{< /imath >}} 中的每一条边，从 {{< imath >}}S{{< /imath >}} 中移除这条边中的任意一个顶点，剩下的集合记为 {{< imath >}}S'{{< /imath >}}。最坏情况每条边都要删掉一个不同的点，所以
{{< math >}}

\left| S' \right| \geq  X-Y

{{< /math >}}
从而
{{< math >}}

\mathbf{E}[\left| S' \right| ]\geq  \mathbf{E}[X]-\mathbf{E}[Y] = np-mp^{3} = f(p)

{{< /math >}}
我们需要选出一个合适的 {{< imath >}}p{{< /imath >}} 来最大化下界 {{< imath >}}f(p){{< /imath >}}，经过简单的求导计算可知在
{{< math >}}

p^{*}=\sqrt{ \frac{n}{3m} }

{{< /math >}}
时 {{< imath >}}\dfrac{\mathrm{d} f}{\mathrm{d}p}=0{{< /imath >}}，带入题设条件 {{< imath >}}m\geq c_{1}n{{< /imath >}}，有
{{< math >}}

p^{*} \leq   \sqrt{ \frac{1}{3c_{1}} }

{{< /math >}}
如果 {{< imath >}}c_{1}\geq \frac{1}{3}{{< /imath >}}，那么 {{< imath >}}p^{*}\leq 1{{< /imath >}}。从而直接取 {{< imath >}}p=p^{*}{{< /imath >}}，带入得到了
{{< math >}}

\alpha(\mathcal{H}) \geq  f(p^{*}) = \frac{2}{3\sqrt{ 3 }} \frac{n\sqrt{ n }}{\sqrt{ m }}

{{< /math >}}
这是取常数 {{< imath >}}c_{2}= \frac{2}{3\sqrt{ 3 }}{{< /imath >}} 即可。

如果 {{< imath >}}c_{1}< \frac{1}{3}{{< /imath >}}，那么利用
{{< math >}}

m\geq c_{1}n \implies \sqrt{ \frac{c_{1}n}{m} } \leq  1

{{< /math >}}
带入 {{< imath >}}p=\sqrt{ \frac{c_{1}n}{m} }{{< /imath >}}，得到
{{< math >}}

\begin{align*}
\alpha(\mathcal{H}) & \geq  f(p) \\
 & = n\left( \sqrt{ c_{1} }\sqrt{ \frac{n}{m} } \right) - m\left( \sqrt{ c_{1} }\sqrt{ \frac{n}{m} } \right)^{3} \\
 & = (\sqrt{ c_{1} }-c_{1}\sqrt{ c_{1} }) \frac{n\sqrt{ n }}{\sqrt{ m }}
\end{align*}

{{< /math >}}
由于 {{< imath >}}c_{1}<  \frac{1}{3}< 1{{< /imath >}}，因此 {{< imath >}}\sqrt{ c_{1} }-c_{1}\sqrt{ c_{1} }>0{{< /imath >}}，从而取 {{< imath >}}c_{2}=\sqrt{ c_{1} }-c_{1}\sqrt{ c_{1} }{{< /imath >}} 即可。
