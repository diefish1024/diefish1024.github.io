## Problem 1

设 {{< imath >}}H = (V, \mathcal{F}){{< /imath >}} 是一个没有孤立点的超图，满足每条超边至少包含 3 个顶点，且任意两条超边恰好共享一个顶点。假设 {{< imath >}}H{{< /imath >}} 不是 2-可着色的（即不能用两种颜色对顶点染色使得每条超边内都有不同颜色的点）。

**(1)**

证明每个顶点 {{< imath >}}v{{< /imath >}} 至少属于 {{< imath >}}\mathcal{F}{{< /imath >}} 中的两条超边。

**证**

假设存在一个顶点 {{< imath >}}v{{< /imath >}} 满足 {{< imath >}}d(v)< 2{{< /imath >}}。由于没有孤立顶点，因此 {{< imath >}}d(v)=1{{< /imath >}}，恰好属于一条超边，记为 {{< imath >}}E{{< /imath >}}。

我们将顶点 {{< imath >}}v{{< /imath >}} 染成红色，将 {{< imath >}}E{{< /imath >}} 中的其他点染成蓝色，再将图中除了 {{< imath >}}E{{< /imath >}} 之外的所有点染成红色。由于任意两条超边恰好共享一个顶点，并且 {{< imath >}}v{{< /imath >}} 只在 {{< imath >}}E{{< /imath >}} 中，所以任意一条超边必然和 {{< imath >}}E{{< /imath >}} 存在蓝色交点，存在颜色不同的点。因此我们就构造出了一个合法的 2-着色方案，与题设矛盾。

因此每个顶点至少属于两条超边。证毕。

**(2)**

证明任意两个顶点 {{< imath >}}u, v{{< /imath >}} 恰好共同属于 {{< imath >}}\mathcal{F}{{< /imath >}} 中的一条超边。

**证**

假设存在两个顶点 {{< imath >}}u,v{{< /imath >}}，它们不共同属于任何一条超边。定义 {{< imath >}}N_{u}{{< /imath >}} 为与 {{< imath >}}u{{< /imath >}} 在同一条超边上的邻居，那么 {{< imath >}}v\not\in N_{u}{{< /imath >}}。

我们构造染色方案，首先将 {{< imath >}}u{{< /imath >}} 染为红色，{{< imath >}}N_u{{< /imath >}} 中的所有点染为蓝色，其余顶点（包括 {{< imath >}}v{{< /imath >}}）染为红色。

首先检查是否存在全红边。经过 {{< imath >}}u{{< /imath >}} 的边包含 {{< imath >}}u{{< /imath >}}（红）和 {{< imath >}}N_u{{< /imath >}} 中的点（蓝），非单色和边。对于不经过 {{< imath >}}u{{< /imath >}} 的任意边 {{< imath >}}E{{< /imath >}}，根据 {{< imath >}}(1){{< /imath >}} 有 {{< imath >}}d(u)>1{{< /imath >}}，{{< imath >}}u{{< /imath >}} 至少被两条边经过，设为 {{< imath >}}L_1, L_2{{< /imath >}}，则 {{< imath >}}E{{< /imath >}} 必与 {{< imath >}}L_1, L_2{{< /imath >}} 分别交于 {{< imath >}}x, y{{< /imath >}}。由于 {{< imath >}}u \notin E{{< /imath >}}，交点 {{< imath >}}x, y{{< /imath >}} 必然都在 {{< imath >}}N_u{{< /imath >}} 中且 {{< imath >}}x \neq y{{< /imath >}}。因此，任何不经过 {{< imath >}}u{{< /imath >}} 的边起初至少包含两个蓝色点，不可能为全红。

其次检查是否存在全蓝边。若存在全蓝边 {{< imath >}}F \subseteq N_u{{< /imath >}}，则 {{< imath >}}u{{< /imath >}} 的度数 {{< imath >}}d(u)=|F| \ge 3{{< /imath >}}，这使得任意不经过 {{< imath >}}u{{< /imath >}} 的边在 {{< imath >}}N_u{{< /imath >}} 中初始至少拥有 3 个蓝色顶点（即 {{< imath >}}u{{< /imath >}} 的每个分支各贡献一个）。由于全蓝边集两两相交，根据几何性质，至多只需将 2 个公共顶点改染红色即可破坏所有全蓝边。因为 {{< imath >}}3 > 2{{< /imath >}}，任何不经过 {{< imath >}}u{{< /imath >}} 的边在染色修正后仍保留至少一个蓝点；而经过 {{< imath >}}u{{< /imath >}} 的边因包含 {{< imath >}}u{{< /imath >}} 且未翻转其所有邻居，也绝非全红，故矛盾成立。

同时检查这时是否新产生了全红边：任何包含 {{< imath >}}w{{< /imath >}} 的新红边，若经过 {{< imath >}}u{{< /imath >}}，则除 {{< imath >}}u, w{{< /imath >}} 外必有第三点属于 {{< imath >}}N_u{{< /imath >}}（仍为蓝）；若不经过 {{< imath >}}u{{< /imath >}}，该边原先至少有两个蓝点，翻转 {{< imath >}}w{{< /imath >}} 后仍剩至少一个蓝点。

综上，我们总能构造出合法的 2-染色，这与题目条件矛盾，故假设不成立，任意两点必须共边。

## Problem 2

证明每一个拥有 {{< imath >}}m{{< /imath >}} 条边的图，都包含一个至少拥有 {{< imath >}}(1 - 1/k)m{{< /imath >}} 条边的 {{< imath >}}k{{< /imath >}}-可着色子图。

**证**

我们对图中每个点独立均匀随机地染成 {{< imath >}}k{{< /imath >}} 种颜色中的一种，记这 {{< imath >}}k{{< /imath >}} 种颜色为 {{< imath >}}\{ 1,2,\dots,k \}{{< /imath >}}，顶点 {{< imath >}}v{{< /imath >}} 的颜色为 {{< imath >}}X_{v}{{< /imath >}}，那么
{{< math >}}

\mathbb{P}(X_{v}=u)=\frac{1}{k}

{{< /math >}}
再对每条边 {{< imath >}}e=(u,v){{< /imath >}} 定义指示变量 {{< imath >}}Y_{e}=\mathbb{I}[X_{u}\neq X_{v}]{{< /imath >}}。从而
{{< math >}}

\mathbb{P}(Y_{e}=0)= \sum_{i=1}^{k} \frac{1}{k^{2}} = \frac{1}{k}

{{< /math >}}
从而
{{< math >}}

\mathbf{E}[Y_{e}] = 1\cdot \mathbb{P}(Y_{e}=1) + 0 \cdot \mathbb{P}(Y_{e}=0) = 1- \frac{1}{k}

{{< /math >}}
当 {{< imath >}}Y_{e}=1{{< /imath >}} 说明一条边两端点颜色不同，可以被保留进需要的 {{< imath >}}k{{< /imath >}}-可着色子图。我们计算最终保留的总边数 {{< imath >}}Y{{< /imath >}}，为所有指示变量之和
{{< math >}}

Y=\sum_{e\in E}Y_{e}

{{< /math >}}
那么
{{< math >}}

\mathbf{E}[Y] = \sum_{e\in E}\mathbf{E}[Y_{e}] = m\left( 1-\frac{1}{k} \right)

{{< /math >}}
由于 {{< imath >}}Y{{< /imath >}} 的取值不可能总是小于 {{< imath >}}\mathbf{E}[Y]{{< /imath >}}，因此必然存在一种方案使得保留下的边数满足
{{< math >}}

Y \geq  \mathbf{E}[Y]=m\left( 1-\frac{1}{k} \right)

{{< /math >}}
由于留下的每一条边端点颜色都不同，因此这是一个 {{< imath >}}k{{< /imath >}}-着色子图。

## Problem 3

设 {{< imath >}}n \ge 4{{< /imath >}} 且 {{< imath >}}t \ge 3\sqrt{n}{{< /imath >}}。证明：对于任意元素互不相同的 {{< imath >}}n \times n{{< /imath >}} 实数矩阵 {{< imath >}}A{{< /imath >}}，我们可以通过重排其列得到一个新矩阵 {{< imath >}}B{{< /imath >}}，使得在 {{< imath >}}B{{< /imath >}} 中没有任何一行包含长度为 {{< imath >}}t{{< /imath >}} 的单调子序列。

**证**

设矩阵 {{< imath >}}A{{< /imath >}} 的列下标集合为 {{< imath >}}\{1, 2, \dots, n\}{{< /imath >}}。我们在所有可能的列排列（共 {{< imath >}}n!{{< /imath >}} 种）中，均匀随机地选取一种排列 {{< imath >}}\sigma{{< /imath >}}，按照该排列重排 {{< imath >}}A{{< /imath >}} 的列得到新矩阵 {{< imath >}}B{{< /imath >}}。

记事件 {{< imath >}}E{{< /imath >}} 为“矩阵 {{< imath >}}B{{< /imath >}} 中至少有一行包含长度为 {{< imath >}}t{{< /imath >}} 的单调子序列”。如果能证明 {{< imath >}}\mathbb{P}(E) < 1{{< /imath >}}，则根据概率法的原理，满足条件的排列必然存在。

题目条件为 {{< imath >}}n \ge 4{{< /imath >}} 且 {{< imath >}}t \ge 3\sqrt{n}{{< /imath >}}。我们首先考察 {{< imath >}}n{{< /imath >}} 较小的情况。当 {{< imath >}}4 \le n < 9{{< /imath >}} 时
{{< math >}}

\sqrt{n} < 3 \implies 3\sqrt{n} > n

{{< /math >}}
由于 {{< imath >}}t \ge 3\sqrt{n}{{< /imath >}}，此时有 **{{< imath >}}t > n{{< /imath >}}**。 在一个仅有 {{< imath >}}n{{< /imath >}} 个元素的行中，显然不可能存在长度为 {{< imath >}}t{{< /imath >}}（且 {{< imath >}}t > n{{< /imath >}}）的子序列。 因此，当 {{< imath >}}n < 9{{< /imath >}} 时，事件 {{< imath >}}E{{< /imath >}} 发生的概率为 {{< imath >}}\mathbf{0}{{< /imath >}}。结论显然成立。

当 {{< imath >}}n\geq 9{{< /imath >}} 时，考虑利用 Union Bound 估计上界。设 {{< imath >}}E_{i}{{< /imath >}} 为第 {{< imath >}}i{{< /imath >}} 行包含长度为 {{< imath >}}t{{< /imath >}} 的单调子序列的事件。那么
{{< math >}}

\mathbb{P}(E) = \mathbb{P}\left( \bigcup_{i=1}^{n}E_{i} \right) \leq  \sum_{i=1}^{n} \mathbb{P}(E_{i}) = n\cdot \mathbb{P}(E_{1})

{{< /math >}}
对于 {{< imath >}}E_{1}{{< /imath >}}，我们考虑从 {{< imath >}}n{{< /imath >}} 个元素选择 {{< imath >}}t{{< /imath >}} 个位置，共 {{< imath >}}\binom{ n }{ t }{{< /imath >}} 中选法。对于选定 {{< imath >}}t{{< /imath >}} 个元素，在 {{< imath >}}t!{{< /imath >}} 种可能的排列中，只有 {{< imath >}}2{{< /imath >}} 种是单调的，从而有上界
{{< math >}}

\mathbb{P}(E_{1}) \leq  \binom{ n }{ t } \frac{2}{t!}

{{< /math >}}
带入就有
{{< math >}}

\mathbb{P}(E) \leq n\cdot \binom{ n }{ t } \frac{2}{t!}

{{< /math >}}
利用不等式 {{< imath >}}\binom{n}{t} \le \frac{n^t}{t!}{{< /imath >}} 对上界进行放缩，可得
{{< math >}}
\mathbb{P}(E) \leq n \cdot \frac{n^t}{t!} \cdot \frac{2}{t!} = \frac{2n^{t+1}}{(t!)^2}
{{< /math >}}
由条件 {{< imath >}}t \ge 3\sqrt{n}{{< /imath >}} 可知 {{< imath >}}n \le \frac{t^2}{9}{{< /imath >}}。将此不等式代入上式，我们将变量统一为 {{< imath >}}t{{< /imath >}}。记该上界函数为 {{< imath >}}f(t){{< /imath >}}，则
{{< math >}}
\mathbb{P}(E) \leq \frac{2(t^2/9)^{t+1}}{(t!)^2} = f(t)
{{< /math >}}

由于 {{< imath >}}n \ge 9{{< /imath >}}，故 {{< imath >}}t \ge 3\sqrt{9} = 9{{< /imath >}}。我们首先验证 {{< imath >}}t{{< /imath >}} 取最小值 {{< imath >}}9{{< /imath >}} 时的情况：
{{< math >}}
f(9) = \frac{2 \cdot (9^2/9)^{10}}{(9!)^2} = \frac{2 \cdot 9^{10}}{(362880)^2} \approx \frac{6.97 \times 10^9}{1.31 \times 10^{11}} \approx 0.053 < 1
{{< /math >}}

接下来考察 {{< imath >}}f(t){{< /imath >}} 随 {{< imath >}}t{{< /imath >}} 增大的变化趋势。利用 Stirling 公式 {{< imath >}}t! \approx \sqrt{2\pi t}(t/e)^t{{< /imath >}} 估算其量级，分母 {{< imath >}}(t!)^2{{< /imath >}} 的增长主要由 {{< imath >}}t^{2t}{{< /imath >}} 主导，而分子的增长主要由 {{< imath >}}(t^2)^t = t^{2t}{{< /imath >}} 以及常数底数因子主导。具体观察其底数比率
{{< math >}}
\frac{n^{t+1}}{(t!)^2} \approx \frac{(t^2/9)^t}{(t/e)^{2t}} = \left( \frac{t^2}{9} \cdot \frac{e^2}{t^2} \right)^t = \left( \frac{e^2}{9} \right)^t
{{< /math >}}
由于 {{< imath >}}e^2 \approx 7.39 < 9{{< /imath >}}，该底数小于 {{< imath >}}1{{< /imath >}}，因此 {{< imath >}}f(t){{< /imath >}} 以指数级速度趋于 {{< imath >}}0{{< /imath >}}。

综上所述，对于所有 {{< imath >}}n \ge 9{{< /imath >}} 且 {{< imath >}}t \ge 3\sqrt{n}{{< /imath >}}，都有 {{< imath >}}\mathbb{P}(E) \le f(t) \le f(9) < 1{{< /imath >}}。这表明在所有 {{< imath >}}n!{{< /imath >}} 种排列中，必然存在一种排列，使得生成的矩阵 {{< imath >}}B{{< /imath >}} 中没有任何一行包含长度为 {{< imath >}}t{{< /imath >}} 的单调子序列。

证毕。

## Problem 4

设 {{< imath >}}v_1, \dots, v_n{{< /imath >}} 是 {{< imath >}}\mathbb{R}^n{{< /imath >}} 中的 {{< imath >}}n{{< /imath >}} 个单位向量。证明存在一组符号 {{< imath >}}\epsilon_i = \pm 1{{< /imath >}}，使得 {{< imath >}}\|\epsilon_1 v_1 + \dots + \epsilon_n v_n\| \le \sqrt{n}{{< /imath >}}，并且证明 {{< imath >}}\sqrt{n}{{< /imath >}} 这个界限是不可改进的。

**证**

首先证明存在性。明存在一组符号 {{< imath >}}\epsilon_i \in \{+1, -1\}{{< /imath >}}，使得 {{< imath >}}\|\sum \epsilon_i v_i\| \le \sqrt{n}{{< /imath >}}。假设 {{< imath >}}\epsilon_1, \epsilon_2, \dots, \epsilon_n{{< /imath >}} 是独立同分布的随机变量，且取值概率满足
{{< math >}}
\mathbb{P}(\epsilon_i = 1) = \frac{1}{2}, \quad \mathbb{P}(\epsilon_i = -1) = \frac{1}{2}
{{< /math >}}
这意味每个 {{< imath >}}\epsilon_i{{< /imath >}} 的期望值为 {{< imath >}}E[\epsilon_i] = 0{{< /imath >}}，方差为 {{< imath >}}E[\epsilon_i^2] = 1{{< /imath >}}。

考察向量和模长的平方。设随机变量 {{< imath >}}X{{< /imath >}} 为
{{< math >}}
X = \left\| \sum_{i=1}^n \epsilon_i v_i \right\|^2
{{< /math >}}
展开得到
{{< math >}}
\begin{aligned} X &= \left\langle \sum_{i=1}^n \epsilon_i v_i, \sum_{j=1}^n \epsilon_j v_j \right\rangle \\ &= \sum_{i=1}^n \sum_{j=1}^n \epsilon_i \epsilon_j \langle v_i, v_j \rangle \end{aligned}
{{< /math >}}
根据期望的线性性，有
{{< math >}}
\mathbf{E}[X] = \sum_{i=1}^n \sum_{j=1}^n \mathbf{E}[\epsilon_i \epsilon_j] \langle v_i, v_j \rangle
{{< /math >}}
我们需要计算 {{< imath >}}E[\epsilon_i \epsilon_j]{{< /imath >}}。分别考虑 {{< imath >}}i=j{{< /imath >}} 和 {{< imath >}}i\neq j{{< /imath >}}，有
{{< math >}}
\mathbf{E}[\epsilon_{i}^{2}]=1,\quad \mathbf{E}[\epsilon_i \epsilon_j] = \mathbf{E}[\epsilon_i] \cdot \mathbf{E}[\epsilon_j] = 0 \cdot 0 = 0
{{< /math >}}
从而
{{< math >}}
\mathbf{E}[X] = \sum_{i=1}^n 1 \cdot \langle v_i, v_i \rangle = \sum_{i=1}^n \|v_i\|^2=n
{{< /math >}}
由于一个随机变量的取值不可能严格大于它的期望，因此执照存在一组特定的符号使得
{{< math >}}
\left\| \sum_{i=1}^n \epsilon_i v_i \right\|^2 \le n
{{< /math >}}
从而就证明了
{{< math >}}
\left\| \sum_{i=1}^n \epsilon_i v_i \right\| \le \sqrt{n}
{{< /math >}}
接着我们证明 {{< imath >}}\sqrt{ n }{{< /imath >}} 是紧的。取 {{< imath >}}\mathbb{R}^{n}{{< /imath >}} 中的一组标准正交基，那么这时无论如何选择符号，模长都恒等于 {{< imath >}}\sqrt{ n }{{< /imath >}}，从而不可能存在比 {{< imath >}}\sqrt{ n }{{< /imath >}} 更小的上界。
